# ToDo List API 

API Rest que tem como função manter as listas de tarefas e tarefas dos usuários. 

## Frameworks utilizados 

* Spring Boot
* Spring Web
* Spring Data Jpa
* Spring Security (Oauth2, JWT)
* Lombok
